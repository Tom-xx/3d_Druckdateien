# Summary

Little brother of MagIt Magsafe Guard (http://www.thingiverse.com/thing:1819423) 

this one is a cable protector for lightning usb charge cables. We know Apple likes sleek design, but there is always tears and cracks near cable-jack connection point. This guard limits the cable flexion, so protects the connection point.  Enjoy!

P.S. - Cable inserting may be a little tricky, be patient, be safe. Use a BLUNT tool to push it in

# Print Settings

Printer: Micromake Kossel Delta D1
Rafts: No
Supports: Yes
Resolution: 0,1 - 0,2
Infill: 100%

Notes: 
Supports are only needed for the tip, use the smallest support as possible, and remove carefully. I printed with PLA and PETG, PETG likes 0.2 layer and 90% flow.
