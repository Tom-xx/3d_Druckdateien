# Summary

This file is a base for the excellent Wave Lamp part: it fits to the cuved geometry of the base and  It has a spiral groove for an LED strip and cabling to be mounted to illuminate the lamp  

The wave lamp part can be found here:  

http://www.thingiverse.com/thing:774456

I added a version of the base without the spiral groove for the LED strip. The STL may be invalid with lots of overlapping triangles however it does seem to slice with no problems on GrabCAD Print and Insight
